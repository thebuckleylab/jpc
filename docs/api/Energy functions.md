# Energy functions

JPC provides three main PC energy functions:

* [jpc.pc_energy_fn()](https://thebuckleylab.github.io/jpc/api/Energy%20functions/#jpc.pc_energy_fn) 
for standard PC networks,
* [jpc.hpc_energy_fn()](https://thebuckleylab.github.io/jpc/api/Energy%20functions/#jpc.hpc_energy_fn) 
for hybrid PC models ([Tscshantz et al., 2023](https://journals.plos.org/ploscompbiol/article?id=10.1371/journal.pcbi.1011280)),
* [jpc.bpc_energy_fn()](https://thebuckleylab.github.io/jpc/api/Energy%20functions/#jpc.hpc_energy_fn) 
for bidirectional PC models ([Oliviers et al., 2025](https://arxiv.org/abs/2505.23415)), and
* [jpc.epc_energy_fn()](https://thebuckleylab.github.io/jpc/api/Energy%20functions/#jpc.epc_energy_fn) 
for error-reparameterised PC ([Goemaere et al., 2025](https://arxiv.org/abs/2505.20137)).
